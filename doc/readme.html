Introduction

Proto language

Proto is a language for programming large networks of simple devices. This language allows the programmer to program the whole network to do complex things, with literally a few lines of code.

MIT Proto

MIT Proto was the first implementation and the origin of the Proto language. It provides a compiler, virtual machine 'kernel' and a simulator.

DelftProto

DelftProto is another implementation of the Proto language. It is currently being developed at the Delft University of Technology. At this time, we are focusing on our virtual machine implementation, which —even tough still under development— is already successfully used on some devices. In the near future, a compiler and simulator will also be provided. We already made significant progress in developing the compiler and its source code will soon be publicly available.

Download

Source Code Repository

DelftProto is hosted on GitHub: https://github.com/delftproto/delftproto. To get the source code, clone our git repository:

git clone git://github.com/delftproto/delftproto.git

If you do not want to use git, you can also download a tarball of the latest commit: https://github.com/delftproto/delftproto/tarball/master.

Documentation

Online

The documentation is available online at http://delftproto.nl/doc/.

From Source

The documentation can be generated from the source distribution by running make in the /doc/ directory. Apart from GNU Make, you need Doxygen, Graphviz Dot and (pdf)LaTeX.
