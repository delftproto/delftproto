Address Class Reference

#include <address.hpp>

Detailed Description

An address of a constant series of bytes in memory.

One of the types that can be stored in Data.

Note:
This is just a wrapper for an Int8 constant pointer, with an explicit constructor. This is useful, because 0 cannot be interpreted as an Address (unlike Int8 *), you have to explicitly say Address(0). This makes something like 
 Data d = 0;
unambiguous. (It's interpreted as a Number, not an Address.)
